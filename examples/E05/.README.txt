\section e05_s2 Compilation:

  With G4:\n
  An extra library with user defined Geant4 based classes has to be build:
  <pre>
  cd E05/geant4
  gmake
  </pre>

\section e05_s3 Macros

  For running example with G3:
  <pre>
  \link      E05/load_g3.C load_g3.C     \endlink - macro to load the libraries to run the example with Geant3
  \link       E05/run_g3.C run_g3.C      \endlink - macro for running example 
  \link E05/g3tgeoConfig.C g3tgeoConfig.C\endlink - configuration macro for G3 with TGeo geometry(default)
  \link     E05/g3Config.C g3Config.C    \endlink - configuration macro for G3 with native geometry
  </pre>

  For running example with G4:
  <pre>
  \link       E05/load_g4.C load_g4.C      \endlink - macro to load the libraries to run the example with Geant4
  \link        E05/run_g4.C run_g4.C       \endlink - macro for running example 
   g4config.in   - macro for G4 configuration using G4 commands (called from g4Config.C)
   g4config2.in  - macro for G4 configuration using G4 commands (called from g4Config2.C)
   g4vis.in      - macro for G4 visualization settings (called from set_vis.C) 
  </pre>

  Common macro (called by run_g3.C/run_g4.C):
  <pre>
  \link E05/set_vis.C set_vis.C\endlink - setting visualization
  </pre>

\section e05_s4 To run example 

  <pre>
  With G3 + TGeo:
  root[0] .x \link E05/load_g3.C load_g3.C\endlink
  root[1] .x \link E05/run_g3.C run_g3.C\endlink

  With G4:
  root[0] .x \link E05/load_g4.C load_g4.C\endlink
  root[1] .x \link E05/run_g4.C run_g4.C\endlink

  With G4 + TGeo navigation:
  root[0] .x \link E05/load_g4.C load_g4.C\endlink
  root[1] .x \link E05/run_g4.C run_g4.C\endlink(\link E05/g4tgeoConfig.C "g4tgeoConfig.C"\endlink);

  </pre>
